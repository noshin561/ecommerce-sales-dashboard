# E-Commerce Sales Dashboard

Built using Tableau Public with real sales data.

## Charts:
- Sales by Category (Bar Chart)
- Sales by Size (Pie Chart)
- Sales by Color (Bar Chart)
- Top Categories (Treemap)
- Sales Trend (Line Chart)

## Tools Used:
- Tableau Public
- CSV Data (9,271 rows)
